(static) copy(target, …objects) → {Object}

Creates a copy of the target object (using the same prototype). All the properties of the target object will be copied to the new object.

A list of objects may also be specified to override the properties of the target object.

Parameters:
Name Type Attributes Description
target Object

A target object.
objects Object <repeatable>

A list of objects to be copied.
Source:
  • obj.js, line 31
Returns:

A new object.

Type
Object
Example
var person = {name: 'jane', age: 20, city: 'Melbourne'};
copy(person, {name: 'bob'}); // {name: 'bob', age: '20', city: 'Melbourne'}
