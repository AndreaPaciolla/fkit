fkit — ALL THE THINGS!

This module mixes in the functions and classes from all the other FKit modules. It's available as a convenience, however if you don't need all of FKit then you can require just the module that you need.

fkit.fn — Core Functions

This module defines basic operations on functions.

apply(f, a) curried

Applies a function to a value.

Discussion

In JavaScript we would normally use the expression f(a) to apply a function to a value. This is very concise, however sometimes it's not very useful. You can actully think of function application as a binary operation.

For example, say we want to fold right-to-left over a list of functions and apply each function to the accumulated value:

F.foldRight(F.apply, 0, [F.add(1), F.div(2), F.mul(3)]);

Parameters

f

A function to be applied.

a

A value.

Return Value

The result of f(a).

Example

function sayHi(a) { return ['Hi', a, '!'].join(' '); }
apply(sayHi, 'John'); // Hi John!

applyRight(a, f) curried

Applies a function to a value.

Discussion

It is the same as apply, however the order of the arguments is flipped. This can be useful in certain situations.

For example, say we want to fold left-to-right over a list of functions and apply each function to the accumulated value:

F.fold(F.applyRight, 0, [F.add(1), F.div(2), F.mul(3)]);

Parameters

a

A value.

f

A function to be applied.

Return Value

The result of f(a).

Example

function sayHi(a) { return ['Hi', a, '!'].join(' '); }
applyRight('John', sayHi); // Hi John!

fkit.list — Working with Lists

FKit treats both arrays and strings as lists: an array is a list of elements, and a string is a list of characters.

Representing strings as lists may be a novel concept for some JavaScript users, but it is quite common in other languages. This seemingly simple abstractions yields a great deal of power: it allows you to apply the same list combinators to both arrays and strings.

all(p, as) curried

Discussion

Determines if all elements in the list of as satisfy the predicate function p.

Parameters

p

A predicate function.

as

A list.

Return Value

A boolean value.

Example

function p(a) { return a > 1; }
all(p, [1, 2, 3]); // false
all(p, [2, 3]); // true
all(p, [3]); // true
function q(a) { return a === 'o'; }
all(q, 'foo'); // false
all(q, 'oo'); // true
all(q, 'o'); // true
